# Adventure Works Business Intelligence & Sales Analysis 🚲

## 📌 Project Overview
Engineered a relational data model by extracting, cleaning, and appending multiple regional sales tables using Power Query Editor. Designed comprehensive dashboards in Power BI and Tableau to monitor critical KPIs, including $29.3M in total revenue, $12M in profit, and 60K+ global orders.

## 🤝 Team & My Contributions
This project was completed collaboratively as part of a 6-person team during my ExcelR program. While the final presentation represents our group's combined efforts, my specific hands-on contributions included:
* **Designing and developing the interactive Tableau dashboard** to visualize key metrics like region-wise sales, profit margins, and top-performing products.
* Assisting with data extraction, cleaning, and appending multiple regional sales tables using Power Query.
* Collaborating on the formulation of SQL queries to analyze YoY sales growth and profit margins.

## 🛠️ Tech Stack
* **Data Modeling:** Power Query Editor
* **Database & Querying:** SQL
* **Data Visualization:** Power BI / Tableau / Excel

## 🚀 Key Deliverables
* Executed advanced SQL queries to analyze YoY sales growth, profit margins, and customer retention, driving actionable insights for seasonal product bundling.
* Engineered a clean relational data model by appending multiple regional sales tables.
